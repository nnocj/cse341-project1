CSE 341 Project 1 - Contact List API

Welcome to the CSE 341 Project 1 - Contact List API. This API allows you to manage a list of contacts.

Endpoints

  • GET all contacts
    • View All Contacts
  • GET one contact by ID
    • View Contact
  • POST a new contact

    • To post, use Postman or custom form.

  • PUT to update a contact
  • DELETE a contact
