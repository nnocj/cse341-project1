CSE 341 Project 1 - Contact List API

Welcome to the CSE 341 Project 1 - Contact List API. This API allows you to manage a list of contacts.

Use the endpoints provided to interact with the contact list.

Endpoints

  • GET '/api/contacts' - Retrieve all contacts
    • View All Contacts
  • GET '/api/contacts/:id' - Retrieve one contact based on ID
